ゲームの概要
　弾幕を避けて、アイテムを回収するゲームです。
　被弾＝ゲームオーバーとなっておりますのでご注意ください。
　アイテムはステージ開始後、一定時間で規定の数が出現します。
　難易度はEasy(かんたん),Normal(ふつう),Hard(むずかしい)の三種類、
　ステージは通常ステージ10個、エクストラステージ2個で構成されています。
　(エクストラステージには複数の難易度が設定されておらずNormalのみが選択可能です。)
　通常ステージでは弾幕ゲームに初めて触れる方でも楽しめるように作ったつもりです
　腕に自信のある方はエクストラステージに挑戦してみてください。

操作説明
　メニュー画面
　　ステージ、難易度の選択 : 十字キー
　　ステージ、難易度の決定 : Enterキー
　　戻る : Esc(エスケープ)キー
　ゲーム画面
　　移動 : 十字キー
　　低速化 : Ctrl(コントロール)キー(左右どちらでも)
　　戻る : Esc(エスケープ)キー

動作環境
  OS
　　Windows系列 : WindowsXP以降(vista,7は未検証)
　　Mac系列 : 未検証、正常に動作しない、起動しない可能性大いにあり
　　Linux系列 : 未検証、Wine使用により正常に動作する可能性
　　iOS、Android等各種スマホOS : 未検証、正常に動作しない、起動しない可能性大いにあり
　CPU等パソコンの性能面
　　CPU : 2GHz程度(これより低い性能でも動作する可能性は十分にある)
　　メモリ : 500MB以上(CPUと同様)
　　ディスプレイ : 横方向に800ドット、縦方向に900ドット以上確保できるもの
　　　　　　　　　 (モニターの向きを90度変えればそれ以下でも動作可能)
		   (足りなくてもプレイは出来ますが、画面の一部が見えなくなるため
		　　不自然かつ難しくなってしまい快適とは言えないと思います。)
　ソフトウェア関連
　　DirectX8以上がインストールされていること

その他
　諸ファイルについて
　　フォルダ内の"packfile"、"hsptmp"、"start.ax"、"hgimg3.as"の書き換えはご遠慮ください。
　　プログラムが正常に動作しなくなることが予想されます。
　　"score.txt"に関しては多少書き換えても問題ないかと思いますが、お勧めはできません。
　　自己責任でお願いします。
　　一文字毎に改行を入れ、0を32個書き込めば、初期状態に戻ります。
　　"もでるのもと4.png"を書き換えて頂くと、弾及び機体のテクスチャが、
　　"枠1_透明化済み.png"を書き換えて頂くと、プレイアブルエリア外のテクスチャが変更できます。
　　アルファチャンネルが設定されておりますので、編集される方はご理解の上お願いします。
　当たり判定について
　　自機の判定は機体のテクスチャの中央部にある水色の部分のみです。
　　弾のテクスチャの最外縁部から2ドットほどめり込めます。

終わりに
　この度は"--弾幕-- AperiTuosOculos"を手に取って頂き、ありがとうございます
　弾幕ゲームが作ってみたかったので制作してみました。
　その際、弾幕ゲームの本質は〜とあれこれ思索した結果、
　自機のショットは不要、エフェクトも邪魔という結論に至りました。
　(既存のゲームを非難している訳ではありません)
　その為、当初は純粋な耐久弾幕集にする予定だったのですが、
　面白みに欠けるという意見が寄せられたので、
　アイテムを取得するという謎の形式に落ち着きました。
　ここにエフェクトやら自機のショットやらを加えて、
　それでいて大規模かつ流麗なものを作られているプログラマーの方々には本当に感服します。
　(私はこの程度でもかなり苦労しました。)
　個人的には弾幕が汚く見えるので、今後機会があればもう少し綺麗な弾幕を作りたいところです。
　ソースも無駄に長く、改善したいとは思っています。

　タイトルの"Aperi Tuos Oculos"はラテン語で"あなたの目を開きなさい"の意(多分)。
　(避け能力的にも、実際に目を見開くという意味でも)開眼するっていう意味になればと思って付けました。
　ラテン語を勉強したことがあるわけではないので解説書を片手に四苦八苦、
　最終的には人の手を借りてこの一文を書きました。

　何にせよ、皆様がこのゲームを楽しんでいただけることを願っております
